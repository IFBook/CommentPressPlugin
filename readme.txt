== Description ==

Commentpress allows readers to comment paragraph by paragraph in the margins of a text. You can use it to annotate, gloss, workshop, debate and more!

== Installation ==

This plugin only works in combination with the Commentpress Theme:
https://github.com/IFBook/CommentPressTheme
Please install and activate the theme before doing the following:

1) Unzip the .zip file into `/wp-content/plugins/commentpress`
2) Activate the plugin
3) Visit settings page and configure plugin
3) You are done!
